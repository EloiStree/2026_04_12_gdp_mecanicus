**P**: Printable

This is not going to be an active repository.
I just want a place to store interesting gear systems that could be fun to visualize in Godot, especially in VR.

The goal is not to simulate accurate physics, but rather to illustrate motion with a bit of code.

It could be useful for your project if you need to create fake gears that run and move in synchronized variations.
